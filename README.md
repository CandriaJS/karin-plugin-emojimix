## 介绍 📝
`柠糖emoji` 是一个 `Yunzai-Bot` | `Karin` 的扩展插件，提供emoji功能。

如有问题请提交 `issue` 或加入 Q 群: `272040396` 📬

## 安装与更新 🔧

在`Karin`目录执行以下命令

```bash
pnpm add @candriajs/karin-plugin-emojimix@latest -w
```
